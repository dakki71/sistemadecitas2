Pasos para tu cita

Paso 1:

Primero, crear una cuenta con tus datos (nombres, apellidos, dni, correo, etc), para acceder a nuestros servicios médicos.

Paso 2:

Luego, elige tu especialidad, el doctor de tu preferencia y el horario que mejor se acomode a tu persona.

Paso 3:

Después, elige tu método de pago (Yape, tarjeta o efectivo), para así confirmar tu cita de manera instantánea.

Paso 4:

Finalmente, preséntate en el hospital en la fecha programada y entrega tu DNI en caja para entrar a tu cita.
